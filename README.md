# Music Recommendation System

This project is developed as part of the Hex Softwares Machine Learning Internship.

## Project Description
A Machine Learning based Music Recommendation System that recommends songs to users based on similarity techniques and user preferences.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## How to Run
1. Clone the repository
2. Install dependencies using `pip install -r requirements.txt`
3. Run `recommendation.py`
